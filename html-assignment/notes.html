Some new things I learned today

Here are some useful tags that I will inevitably refer back to in the not-so-distant-future:

img
a.k.a. the image element; used to link an image to the HTML
src
a.k.a. source; an attribute that tells the image tag which image it should display
alt
a.k.a. alternate text; provides description of the image for search engines, screen readers and in case the image does not load.
< a >
a.k.a. anchors; used to link everything to your website including pages, photos, etc.
target="_blank"
used to ensure your link opens in a new URL tab
